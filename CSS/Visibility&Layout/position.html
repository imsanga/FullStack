Static

Note: The elements will be in regular document flow

Relative

Note: we can perform offset(top,left,bottom,right) in static

Absolute

Note: The elements go outside the regular document flow, it checks any non-static positions are there or not in parent, if not - it aligns with root element. The remaining elements takes up it's space

Fixed

Note: The elements get fixed in the document flow, it get aligns to the root element and we can perform offset to align the element

Sticky

Note: The elements get sticky in the document flow, it get aligns to the root element(eg:where it sticks when the element reaches Top: 0px) and we can perform offset to align the element
